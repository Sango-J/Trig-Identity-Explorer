# **App Name**: Trig Identity Explorer

## Core Features:

- Identity Input: Allow users to input a trigonometric identity they want to verify.
- Step-by-Step Verification: Show a detailed, step-by-step process to verify if the identity is true or false, powered by a tool using an LLM for symbolic manipulation.
- Interactive Manipulation: Enable users to interact with the steps, potentially changing approaches to see different verification paths.
- Trigonometric Function Reference: Provide a comprehensive table explaining each trigonometric function and its properties for easy reference.
- Identity Examples: Offer a list of pre-loaded Pythagorean, Sum & Difference, Double Angle, and Half Angle trigonometric identities.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to represent knowledge and precision.
- Background color: Light gray (#EEEEEE) for a clean and unobtrusive backdrop.
- Accent color: Orange (#FF9800) to highlight interactive elements and important steps.
- Body and headline font: 'Inter' for a clean, modern, and readable interface.
- Use clear and intuitive icons to represent different functions and operations.
- Maintain a clear, logical layout with a focus on readability and ease of navigation.
- Use subtle transitions and animations to guide users through the verification steps.
